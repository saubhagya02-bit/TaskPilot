# 📌 TaskPilot – Task Management Web App

TaskPilot is a full-stack task management web application that allows users to manage their daily tasks efficiently with authentication, dashboard insights and CRUD operations.

---

## 🚀 Features

### 🔐 Authentication
- User Registration (Signup)
- User Login
- JWT Token-based Authentication
- Protected Routes using Auth Guard

### 📊 Dashboard
- User profile section (Name + Profile UI)
- Sidebar navigation
- Clean and modern layout

### ✅ Task Management
- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Add due date for tasks
- Display tasks with status and date

---

## 🛠️ Tech Stack

### Frontend
- Angular (Standalone Components)
- TypeScript
- HTML5 + CSS3
- Angular Router
- FormsModule
- ngx-toastr (notifications)

### Backend
- Node.js
- Express.js
- MySQL Database
- JWT Authentication
